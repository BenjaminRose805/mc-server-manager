### Integrations

#### Integration
- **Description:** ElectronAPI interface types the window.electronAPI object exposed by Electron's preload.ts via contextBridge. Frontend code accesses it as window.electronAPI?.method() with full type safety. The interface is optional (electronAPI?) so browser mode correctly sees it as possibly undefined.
- **Frontend Component:** window.electronAPI (global type augmentation)
- **Backend Endpoint:** Electron IPC via preload.ts contextBridge
- **Data Flow:** Frontend code → window.electronAPI?.method(args) → ipcRenderer.invoke(channel) → ipcMain.handle(channel) → auth/launcher modules
